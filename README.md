**Customer-Behaviour-Analysis**
End to end data analytics project showcasing customer purchase behavior using Python, SQL and Power BI.

**Overview**
This project showcases an end-to-end data analytics solution, from raw data ingestion to business-ready insights. It highlights hands-on experience in Python, SQL, and Power BI, focusing on data cleaning, analysis, visualization, and reporting to support data-driven decision-making.

**Dataset**
1. Structured dataset in CSV/Excel format
2. Used to analyze trends, KPIs, and performance metrics
3. Prepared and optimized for analysis and visualization

**Tools & Technologies**
1. Python (Pandas, NumPy) – Data loading, EDA, and cleaning
2. SQL (PostgreSQL / MySQL / SQL Server) – Data analysis and querying
3. Power BI – Interactive dashboards and KPIs
4. Gamma – Executive-level presentation
5. Jupyter Notebook – Analysis workflow

**Key Steps**
1. Performed Exploratory Data Analysis (EDA) to identify trends and anomalies
2. Cleaned and transformed data for accuracy and consistency
3. Ran optimized SQL queries to answer business questions
4. Built a Power BI dashboard with slicers and drill-downs
5. Delivered insights through a structured report and presentation

**Results**
1. Improved data quality and reporting accuracy
2. Identified key trends and actionable insights
3. Created a stakeholder-ready dashboard for quick decision-making

**Skills Demonstrated**
1. Data Analysis & Visualization
2. SQL Query Optimization
3. Dashboard Design (Power BI)
4. Business Insights & Reporting
